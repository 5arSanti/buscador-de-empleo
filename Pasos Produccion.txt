Frontend
    Teniendo en cuenta que el codigo fuente del Frontend de BUE no esta en QA ni PROD, es necesario conocer la url de alojamiento de la build del Frontend
    Respecivamente se va a realizar una build desde DESARROLLO para que sea cargada tanto a QA y PROD en la carpeta dist.

Backend
    En el archivo app.properties.ini se encuentran las variables de entorno necesarias para la conexion a Base de Datos y para la estructura de API,
    y pueden ser cambiadas directamente en el archivo

    Para poner en marcha el Servidor es necesario ejecutar desde una Consola PuTTY el siguiente comando:
        npm run prod

    Verificar que la conexion a la base de datos es correcta.
    Este comando se ejecuta en la carpeta donde se encuentra el codigo fuente del backend y donde se encuentre el archivo package.json

    NOTA:
        Cada vez que realiza un cambio en el archivo app.properties.ini es necesario reiniciar el servidor
