Paso 1: En la carpeta raiz del proyecto ejecutar el comando:
    git pull origin main

Paso 2: En la carpeta Backend y Frontend ejecutar el siguiente comando:
    npm i

Paso 3: En la carpeta backend ejecutar el siguiente comando y verificar la conexion a base de datos correctamente:
    npm run prod

Paso 4: En la carpeta Frontend ejecutar el siguiente comando y verificar que se creo la carpeta dist:
    npm run build

    NOTA: La version compilada del Proyecto se escuentra en la carpeta "dist", por lo esta carpeta se debe subir junto a la carpeta backend al servidor de produccion

Paso 5: Verificar el dominio en el cual se va a montar el proyecto:
    Ejemplo: "http://domainexample.com"

    Siguiente a esto, se debe agregar este dominion al array "whiteList" ubicado en el archivo index.js en la carpeta backend
    NOTA: "Verificar conexion con el backend desde el frontend para mantener la integridad del proyecto", y "Verificar que e dominio se encuentra dentro de la whitelist para satisfacer la autenticacion de CORS"

Paso 6: Verificar que el proyecto funciona correctamente.
